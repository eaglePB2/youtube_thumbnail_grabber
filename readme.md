# Youtube縮略圖截取器
- 一個簡單的網頁，用來協助我獲取我影片所有縮略圖的大小。
- 通過這個工具，可以幫我監視我的縮略圖是否適合運用在所有的情況。
- 除此之外，也可以通過這個獲取Youtube自動生成的縮略圖來進行對比。
